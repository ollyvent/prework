HTML

  • the head element contains information about the webpage
  • the body element represents the visible information shown to the user
  • the actual webpage information is enclosed within the title tag within the head tag
  • the h1, h2, h3, h4, h5 elements are used to add header topics to the content in the body
  • the h elements reduce increamentally towards h5
  • the img tag is used to add images to the project, it may be by absolute source, relative source or a refference

  • git status is used to check the current branch and status of our code
